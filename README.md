# Asia Webs Guide

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要存放各类独立的 HTML 页面，这些页面可能涉及信息展示、工具、参考资源等不同用途。每个页面均为独立文件，彼此之间没有强关联。

仓库本身不针对任何特定域名或网站，仅作为 HTML 文件的集合与存档点。

## 目录说明

```
/
├── index.html          # 仓库主页（如有）
├── pages/              # 页面存放目录
│   ├── example1.html
│   └── example2.html
└── assets/             # 静态资源（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：存放主要 HTML 页面文件。
- `assets/`：存放页面依赖的静态资源。
- 根目录下的 `index.html` 为可选的主页文件。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可直接通过浏览器打开。
- 部分页面可能引用 `assets/` 目录下的资源，建议保持相对路径不变。
- 页面之间不共享数据或状态。
- 归档内容可能不定期更新或移除。

## 维护说明

- 欢迎提交 Issue 或 Pull Request，但请确保内容与本仓库定位一致。
- 不建议在本仓库中存放与 HTML 页面无关的文件。
- 仓库维护者会定期检查页面可用性，但不对第三方引用的资源负责。

## 许可

本仓库内容默认采用 MIT 许可证，除非页面内另有声明。
